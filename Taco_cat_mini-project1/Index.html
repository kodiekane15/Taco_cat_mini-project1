Taco Cat!

The Taco Cat mini-project is based on the more traditional "Reverse a string" exercise but ultimately determines if a word or phrase is a palindrome.

  • The Reverse a string exercise in some form has been used by Tech companies such as Amazon, Microsoft, VMware, and Norton to assess a dandidate's ability to code an impromptu algorithm
  • Technologies used: HTML, JavaScript, jQuery, CSS, and Bootstrap 4
